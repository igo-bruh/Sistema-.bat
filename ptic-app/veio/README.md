# PTIC App

a ideia aqui eh usar o tarui para fazer uma extensão do menu bat que foi proposto nas aulas de ptic, bat sozinho e feio pra carai e vai passar uma vibe broxa pro trabalho e isso aqui vai ser um diferecnial

## O que ele faz

- Não é só um menu básico no prompt
- É um app com interface que chama comandos/batch por trás
- Serve pra mostrar algo mais profissional pro trabalho

## Por que tem valor

- Evita aquele visual velho do terminal
- Deixa mais fácil de usar e mais apresentável
- Continua usando o batch como lógica/integração

## Estrutura rápida

- `src/` — frontend web (HTML, CSS, JS)
- `src-tauri/` — parte desktop/execução nativa
- `package.json` — dependências e scripts
- `README.md` — é este arquivo mesmo

## Como usar

1. Instalar dependências
2. Rodar o app em modo dev
3. Compilar para gerar um executável

> É basicamente um projeto que transforma o batch em app, dando uma cara mais moderna bglh.
